# **App Name**: SmartNotes

## Core Features:

- Note Creation: Create new notes with a title and body.
- Note Display and Editing: View and edit existing notes.
- AI Formatting Tool: Automatically format note content to improve readability and structure; the LLM will use reasoning to decide when or if to incorporate markdown formatting into the output of the tool.

## Style Guidelines:

- Primary color: Dark blue (#2E3A87) for a professional and modern feel.
- Background color: Light gray (#F0F2F5) to provide a clean and calm backdrop.
- Accent color: Cyan (#4ACBEB) to highlight interactive elements and important actions.
- Body and headline font: 'Inter' sans-serif for a modern and neutral aesthetic.
- Use simple, line-based icons for a clean interface.
- Subtle transitions and animations to enhance user experience when opening, closing, and saving notes.
